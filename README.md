# ビジネスモデル・キャンバス エディター

ビジネスモデルの構築と可視化を支援する，ブラウザベースの対話型エディターです．直感的な付箋操作と多様なテーマ設定により，アイデアの具体化をサポートします．

![2026-01-18_01h22_08](https://github.com/user-attachments/assets/da44823e-5ceb-417d-924b-c3d56171f50d)


## 主な機能

* **直感的な付箋操作**:
* **新規作成**: セクション内をダブルクリックすることで，即座に付箋を追加できます．
* **ドラッグ&ドロップ**: 付箋を別のセクションへ自由に移動可能です．
* **編集**: 付箋のダブルクリックでテキストを編集，右下のドラッグでサイズを変更できます．


* **高度なカスタマイズ**:
* **コンテキストメニュー**: 付箋を右クリックすることで，色（5色），形状（四角・円形），フォント（ゴシック・明朝・手書き風）を個別に変更可能です．
* **全体テーマ**: 「パストラル」「ゲーミングPC」「金属質」など，作業環境や気分に合わせた4種類の外観テーマを選択できます．


* **データ管理**:
* **自動保存**: ブラウザのローカルストレージを使用し，作業内容を自動的に保持します．
* **インポート/エクスポート**: 作成したキャンバスをJSON形式で保存・読み込みが可能です．「名前を付けて保存」では，キャンバスのタイトルがファイル名に反映されます．



## 構成セクション

本アプリは，ビジネスモデル・キャンバスの標準的な9つの要素を網羅しています．

1. **キーパートナー (KP)**
2. **主な活動 (KA)**
3. **主なリソース (KR)**
4. **価値提案 (VP)**
5. **顧客との関係 (CR)**
6. **チャネル (CH)**
7. **顧客セグメント (CS)**
8. **コスト (C$)**
9. **収入 (R$)**

## 使用方法

1. `index.html`（本ファイル）を任意のWebブラウザで開きます．
2. **タイトル編集**: 上部のタイトルをダブルクリックすると，プロジェクト名に変更できます．
3. **付箋の追加**: 各セクションの空きスペースをダブルクリックします．
4. **付箋の編集**: 付箋をダブルクリックして文字を入力します．
5. **付箋の装飾**: 付箋を右クリックし，メニューから色や形を選択します．
6. **保存**: 「名前を付けて保存」ボタンを押すと，ローカルPCにJSONファイルとしてバックアップが可能です．

## 技術スタック

* **Frontend**: HTML5, CSS3 (Flexbox/Grid), JavaScript (Vanilla JS)
* **Storage**: Web Storage API (LocalStorage)
* **Design**: CSS Variables による動的テーマ切り替え
